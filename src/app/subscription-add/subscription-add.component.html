Subscription - nice trick

Subscription can be put together and then a programmer just need to unsubscribe only the one.

Code

        
run() {
  this.subscription = Observable.interval(500)
    .subscribe(value => this.result = value);

  const childSubscription = Observable.interval(1000)
    .subscribe(value => this.childResult = value);

  this.subscription.add(childSubscription);
}

unsubscribe() {
  if(this.subscription) {
    this.subscription.unsubscribe();
  }
}
