Generator and iterator

special kind of function, which can be exited and leter can be re-entered in the last execution point.

Code

      
const fibonnaci = generate_fibonnaci();
setInterval(
  () => this.result += fibonnaci.next().value + " ",
  1000,
);

*generate_fibonnaci() {
  let a = 0, b = 1;
  while (true) {
    yield a;
    const tmp = b;
    b = a + b;
    a = tmp;
  }
}
