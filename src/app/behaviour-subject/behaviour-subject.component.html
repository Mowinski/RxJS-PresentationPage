BehaviorSubject

It is a simple subject which keep last value for new subscribers

Code

      
this.myAge = 18;

runBehaviorSubject() {
  this.behaviorSubject = new BehaviorSubject(this.myAge);
  this.behaviorSubject.subscribe((age: number) => this.result = age);
}
