Observable - time based event

represents the idea of an invokable collection of future values or events.
Single time/value Multiple times/values
Deterministic time Function Iterator
Unknown time Promise Observable

Code

        
const observable = Observable.create((observer) => {
  setTimeout(() => observer.next(1), 1000);
  setTimeout(() => observer.next(2), 2000);
  setTimeout(() => observer.next(3), 3000);
  setTimeout(() => observer.next(5), 5000);
  setTimeout(() => observer.next(8), 8000);
  setTimeout(() => observer.next(13), 13000);
  setTimeout(() => observer.complete(), 21000);
}).debounceTime(1500);

this.$subscription = observable.subscribe((number) => this.result += number + ' ');
